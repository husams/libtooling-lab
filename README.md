# LibTooling Lab

Hands-on lab for learning **Clang LibTooling** (C++) — build linters,
refactoring tools, and code generators on top of Clang's AST.

➡️ **Start here:** [`docs/README.md`](docs/README.md)

- Lab parts & TOC: [`docs/README.md`](docs/README.md)
- Progress checklist: [`docs/PROGRESS.md`](docs/PROGRESS.md)
- How an agent should present this lab: [`CLAUDE.md`](CLAUDE.md)

Runs **fully locally on macOS** against Homebrew LLVM
(`brew install llvm cmake ninja`). Setup is explained in Part 1; the
hands-on work happens in the two starter projects under
[`skeletons/`](skeletons/).
